## How to Use

### Getting Started
1. Open `dont-waste-your-life-app.html` in any web browser
2. No internet connection required - works completely offline
3. No installation needed - it's a single HTML file

### Reading and Studying
1. Select a chapter from the dropdown menu
2. Adjust text size using the A buttons (small, medium, large)
3. Read through the chapter at your own pace
4. Click "Skip to End" to jump to reflection questions
5. Type your answers in the text boxes
6. Answers save automatically as you type

### Emailing Your Work
1. Complete the reflection questions for a chapter
2. Click the "üìß Email All Answers" button
3. Your default email app opens with a pre-formatted message
4. Send to yourself for safekeeping or to a mentor/study group

### Switching Chapters
- Use the chapter dropdown to move between chapters
- Each chapter maintains its own set of saved answers
- You can return to any chapter and pick up where you left off
