# X-Wing Miniatures Game Helper

A comprehensive web-based helper application for playing X-Wing Miniatures Second Edition (2.5). This app guides players through each phase of the game with step-by-step instructions, rules references, and helpful tools.

## Features

- **Phase Tracking**: Visual tracker showing current phase and round
- **Step-by-Step Guidance**: Click buttons to see detailed instructions for any action
- **Dice Roller**: Built-in dice roller for attack and defense rolls
- **Scenario Support**: All 4 standard scenarios with objectives and scoring rules
- **Quick Reference**: Instant access to commonly used rules
- **Experience Levels**: Adapts to beginner, intermediate, or expert players
- **Complete Rules**: Comprehensive coverage of all game phases and mechanics

## Installation

1. Download all files to a folder on your computer:
   - `index.html`
   - `styles.css`
   - `gameData.js`
   - `gameLogic.js`
   - `uiController.js`
   - `app.js`

2. Keep all files in the same folder

3. Open `index.html` in your web browser

That's it! No installation or server required.

## How to Use

### Setup
1. Open the app in your browser
2. Select number of players (2 players standard)
3. Choose which scenario you're playing
4. Select your experience level
5. Click "Start Game"

### During the Game
1. The app shows which phase you're in with a visual tracker
2. Click any action button to see detailed instructions
3. Use the "Next Phase" button to advance through the game
4. Use built-in tools:
   - **Dice Roller**: Roll attack and defense dice
   - **Scenario Info**: See current scenario objectives
   - **Quick Reference**: Access common rules instantly

### Navigation
- **Next Phase** button: Advance to next phase
- **Previous Phase** button: Go back if needed
- **Quick Reference** button: See all rules at a glance
- **Reset Game** button: Start over from setup

### Keyboard Shortcuts
- `Escape`: Close detail panels
- `Right Arrow`: Next phase
- `Left Arrow`: Previous phase

## What's Included

### Game Phases
- **Planning Phase**: Set maneuver dials, determine first player
- **System Phase**: Resolve system abilities, drop devices
- **Activation Phase**: Ships move and perform actions
- **Engagement Phase**: Ships attack
- **End Phase**: Remove tokens, recover charges, check victory

### Scenarios
- Assault at the Satellite Array
- Chance Engagement
- Salvage Mission
- Scramble the Transmissions

### Rules Coverage
- Complete attack sequence
- All actions (Focus, Evade, Lock, Boost, Barrel Roll, etc.)
- Advanced maneuvers (K-turns, Tallon Rolls, etc.)
- Stress rules
- Overlapping ships
- Obstacles
- Range and arc rules
- Damage and destruction
- Victory conditions

## Tips for Best Experience

1. **Use on a tablet or laptop** at the gaming table for easy reference
2. **Keep the app open** on a second screen while playing
3. **Bookmark frequently used sections** by opening them often
4. **Use the dice roller** to save time and reduce physical dice handling
5. **Check scenario info** regularly to remember objectives

## Troubleshooting

**App won't load?**
- Make sure all 6 files are in the same folder
- Try a different browser
- Make sure JavaScript is enabled

**Buttons not working?**
- Refresh the page
- Check browser console for errors (F12)
- Make sure you clicked "Start Game" after setup

**Want to restart?**
- Click "Reset Game" button
- Or refresh the page
